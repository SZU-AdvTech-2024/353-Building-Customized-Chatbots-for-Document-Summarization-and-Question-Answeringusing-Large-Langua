基于 Qwen1.5 1.8B大语言模型与 Langchain 等应用框架实现

### 1. 环境配置

+ 首先，确保你的机器安装了 Python 3.8 - 3.11 

```
$ python --version
Python 3.11.7
```

接着，创建一个虚拟环境，并在虚拟环境内安装项目的依赖

```shell
# 安装依赖
$ pip install -r environment.txt
```

### 一键启动

按照以下命令启动项目

```shell
$ python startup.py -a
```
